# ⏱️ Time Tracker App

A productivity-focused mobile application built with Flutter that helps users manage tasks, track work sessions, analyze productivity, and stay motivated through a rewards system.

---

## 🚀 Features

- ✅ Task Management
- ⏱️ Time Tracking
- 📊 Productivity Analytics
- 🎯 Reward System
- 💾 Offline Data Storage
- 📈 Interactive Charts & Statistics
- 📅 Session History Tracking
- 🎨 Clean Material Design UI
- ⚡ Fast & Lightweight Performance

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|----------|
| Flutter | Cross-platform mobile development |
| Dart | Programming language |
| Riverpod | State Management |
| Hive | Local Database |
| FL Chart | Data Visualization |
| Intl | Date & Localization Support |
| Material Design | User Interface |

## 🖥️ Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/time_tracker_app.git
```

### Navigate to project folder

```bash
cd time_tracker_app
```

### Install dependencies

```bash
flutter pub get
```

### Run the application

```bash
flutter run
```
